# Data Growth Engine

Production-ready marketing website for Data Growth Engine (DGE), built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Deploying To Vercel

1. Push this repo to GitHub.
2. In Vercel, import the repository.
3. Vercel should detect `Next.js` automatically.
4. Deploy.

No environment variables are required for the current version.
